#### Versioning Syntax:

The versioning syntax for AreaResetterPro consists of **three parts:**

**Example:** ```1.4.0.27-PAPER-1.21```

1. ```1.4.0.27``` (MAJOR.MINOR.PATCH.BUILD)
2. ```PAPER```    (CORRESPONDING SERVER FORK)
3. ```1.21```     (CORRESPONDING MINECRAFT VERSION)
